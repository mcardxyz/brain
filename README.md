# SecondBrain
___
Usage is very simple:

🧩 **PARA**
- Action-focused.
- **Projects** for deadlines
- **Areas** for ongoing pursuits
- **Resources** for all reference materials
- **Archive** for completed work.

💡 **ZETA**
- Idea-centric.
- **Permanent Notes** for lasting insights, for research
- **Fleeting Notes** for quick ideas.

📅 **DAILY**
- Organized reflections on a daily, weekly, or monthly basis.

💭 **STICKY**
- Temporary notes for brainstorming.

⚙️ **SYSTEM**
- Core vault setup, housing media, templates, and configurations.



___

*Don’t stray from a working system. Don’t let anything stop you from growing your knowledge base.*
