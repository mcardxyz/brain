# Microsoft Defender

## Setting up a Microsoft Defender Admin role
1. Go to https://security.microsoft.com
2. On the left menu, go to *Permissions*
3. Under "Microsoft Defender XDR", click *Roles*
4. Select *+ Create custom role*
5. Set up the basics:
	1. Role name: `Defender Admin Role`
	2. Description: . . .
6. Choose permissions:
	1. Security operations:
		1. *All read and manage permissions*
	2. Security posture:
		1. *All read and manage permissions*
	3. Authorization and settings
		1. *All read and manage permissions*
7. Click *Next*, and *+ Add assignment*. Assign the user or group
8. Select the *Data and scope* that the user has access to
9. *Next* + *Submit* !


## Onboarding to manage devices using Defender for Endpoint
1. Go to https://security.microsoft.com
2. On the left menu, go to *Settings* → *Endpoints*
3. *Configuration management* → *Onboarding*
